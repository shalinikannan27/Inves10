## Project Overview
Inves10 is a beginner-friendly micro-investment platform designed to help first-time users start investing with minimal amounts. The platform focuses on simplicity, transparency, and trust, making it suitable for users from rural, semi-urban, and low-income backgrounds.

The application provides an intuitive onboarding flow, basic authentication, guided investment recommendations, wallet tracking, and transaction history to help users understand how small investments can grow over time without overwhelming them with complex financial jargon.
